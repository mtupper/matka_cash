## Getting Started

### Prerequisites

- Rust and Cargo
- Solana CLI
- Anchor Framework
- Node.js and Yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/matka_cash.git
cd matka_cash
```

2. Install dependencies:
```bash
yarn install
```

3. Build the program:
```bash
anchor build
```

4. Test the program:
```bash
anchor test
```
